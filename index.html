HTML (HyperText Markup Language):
HTML is the standard language used to create the structure and content of web pages.
It defines elements like headings, paragraphs, links, images, and more.

CSS (Cascading Style Sheets):
CSS controls the visual appearance of web pages including layout, colors, fonts, and spacing.
It allows you to style HTML elements and make your site look attractive and responsive.
